| Exam | Level | Significance |
|------|-------|-------------|
| **ACESSO** | A1 | First official milestone |
| **CIPLE** | A2 | Required for Portuguese citizenship/permanent residency |
| **DEPLE** | B1 | Independent user |
| **DIPLE** | B2 | Required for university admission in Portugal |
| **DAPLE** | C1 | Academic/professional fluency |
| **DUPLE** | C2 | Near-native mastery |

## What Each Exam Tests

### ACESSO — A1
| Component | Duration | Weight |
|-----------|----------|--------|
| Reading Comprehension | 20 min | 20% |
| Written Production & Interaction | 20 min | 15% |
| Listening Comprehension | 25 min | 35% |
| Spoken Production & Interaction | 8–10 min | 30% |

- Reading: signs, short notices, forms, labels
- Writing: filling forms, very simple informal texts
- Listening: personal details, basic directions, simple announcements
- Speaking: personal identification, basic public topics (done in pairs)

### CIPLE — A2
| Component | Duration | Weight |
|-----------|----------|--------|
| Reading + Writing combined | 1h 15m | 45% |
| Listening | 30 min | 30% |
| Speaking | 15 min | 25% |

- Listening: social exchanges, directions, accommodation, prices, health, phone messages
- Speaking: personal info, directions, hotel booking, ordering food, personal experiences

### DEPLE/DIPLE/DAPLE/DUPLE — B1–C2
From B2 onwards adds **Structural Competence** (grammar/structure tasks).

## Official Curriculum Reference: Referencial Camões PLE (2017)

**Free PDF:** https://www.instituto-camoes.pt/images/REFERENCIAL_ebook.pdf

This is the official Portuguese CEFR profile — grammar, vocabulary, and pragmatic functions per level A1–C2. The primary alignment document for our curriculum.

Organised into:
1. **Pragmatic Component** — communicative functions per level
2. **Notional Component** — conceptual categories per level
3. **Linguistic Component** — grammar, vocabulary, pronunciation per level

## Grammar Progression by Level

| Level | Core Grammar |
|-------|-------------|
| A1 | Present tense (regular + key irregulars: ser, estar, ter, ir, fazer), noun/adjective gender agreement, definite/indefinite articles, subject pronouns, basic prepositions, numbers, time |
| A2 | Pretérito Perfeito Simples (simple past), near future (ir + infinitive), reflexive verbs, comparatives/superlatives, direct/indirect object pronouns |
| B1 | Imperfeito (imperfect), Pretérito Mais-que-Perfeito, Conjuntivo (subjunctive) introduced, personal infinitive, passive voice |
| B2 | Subjunctive (present/imperfect/future), conditional, complex relative clauses, reported speech, Structural Competence component added |
| C1 | Compound tenses, aspectual distinctions, advanced nominalization, register variation |
| C2 | Full mastery, stylistic precision, archaic/literary forms |

## A1 Priority Vocabulary (from Referencial)

- Greetings: Bom dia, Boa tarde, Boa noite, Olá, Oi
- Politeness: por favor, se faz favor, desculpe, obrigado/a, de nada
- Basic responses: Sim, Não, Tudo bem?, Com licença
- Survival: Não entendo. Pode repetir? Fala mais devagar, por favor.
- Self-introduction: O meu nome é... / Eu sou... / Sou de...
- Seasonal: Bom fim de semana!, Feliz Natal!, Parabéns!
